# 题目

## 相同的树

### 来源:

[力扣-相同的树](https://leetcode-cn.com/problems/same-tree/)

### 题目内容

给定两个二叉树，编写一个函数来检验它们是否相同。

如果两个树在结构上相同，并且节点具有相同的值，则认为它们是相同的。

### 示例?1:

```plaintext
输入:       1         1
          / \       / \
         2   3     2   3

        [1,2,3],   [1,2,3]

输出: true
```

### 示例 2:

```plaintext
输入:      1          1
          /           \
         2             2

        [1,2],     [1,null,2]

输出: false
```

### 示例?3:

```plaintext
输入:       1         1
          / \       / \
         2   1     1   2

        [1,2,1],   [1,1,2]

输出: false
```
