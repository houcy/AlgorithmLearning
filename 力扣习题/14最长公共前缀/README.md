# 题目

## 最长公共前缀

### 来源:

[力扣-最长公共前缀](https://leetcode-cn.com/problems/longest-common-prefix)

### 题目内容

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 `""`。

### 示例 1:

> 输入: ["flower","flow","flight"]<br>
> 输出: "fl"

### 示例 2:

> 输入: ["dog","racecar","car"]<br>
> 输出: ""<br>
> 解释: 输入不存在公共前缀。

### 说明:

所有输入只包含小写字母 `a-z` 。
