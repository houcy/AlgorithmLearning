## 第k个排列

### 来源:

[力扣-第k个排列](https://leetcode-cn.com/problems/permutation-sequence/)

```plaintext

"132"
"213"
"231"
"312"
"321"
```

给定?`n` 和?`k`，返回第?`k`?个排列。

#### 说明：

- 给定 n?的范围是 [1, 9]。
- 给定 k?的范围是[1, ?n!]。

### 示例?1:

```plaintext
输入: n = 3, k = 3
输出: "213"
```

### 示例?2:

```plaintext
输入: n = 4, k = 9
输出: "2314"
```
